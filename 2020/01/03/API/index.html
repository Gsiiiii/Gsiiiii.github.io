什么是API?
- API（，应用程序接口）是一些预先定义的函数，或指软件系统不同组成部分衔接的约定。 目的是提供应用程序与开发人员基于某软件或硬件得以访问一组例程的能力，而又无需访问原码，或理解内部工作机制的细 —————–<<百度百科>>

  • String (字符) – java.lang.String
    • String,StringBuilder(不安全),StringBuffer(安全)
      • 三者的区别:
        • String 字符串是不可变的
        • 在修改字符串操作比较多的时候用 StringBulider&StringBuffer
        • 在要求线程安全的情况下用StringBuffer是线程安全,它的执行效率比StringBulider 要低
        • 在不要求线程安全的情况下用StringBulider,StringBuilder是线程不安全的,它的执行效率比StringBuffer 要高
  • 集合
    • Collection
      • List
      • Set
