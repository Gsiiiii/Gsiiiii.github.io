如何简单快速的创建一个属于自己的Blog

  • 第一步 :

    你需要在Github上注册一个属于你自己的 “Gayhub” 直达地址–世界上最大的同性交友平台

  • 第二步 :

    • 登录注册的账户,然后创建一个属于自己的仓库

      PS:<如果我的用户名名叫Gsiiiii 创建的仓库名就是 Gsiiiii.github.io>

    • 进入设置

  • 第三步 需要下载的工具:

    • 接下来就是开始下载创建博客的工具

      • 我们会用到几个需要的软件

      • Git – 一个 开源 的分布式版本控制系统

      • Git这只是个版本管理工具,它和Github是建立在Git上的一个网站(一个用于让程序员交流的平台),这是两个不同的东西

      • Git “创始人” 是 Linus Torvalds 也是Linux系统之父 他为了帮助并管理 Linux 内核的开发而开发的的一个开放源代码的版本控制软件


      • Hexo–成立于 2012年 这是一个快速帮助你搭建网站的的 框架


      • Node-Js– 是一个基于Chrome v8 引擎的JavaScript 的运行环境


    • 获取SSH(将Git和github关联起来)

      • 打开Git 当 Git安装完成的时候,在桌面和任意文件中鼠标 右击 都会出现

        Git GUI Here 和 Git Bash Here 两个选项

        我们选择下面的 Git Bash Here

    • 下载并安装 Git Node / hexo( 需要命令行安装 )

      • 在桌面新建一个文件夹 ,命名无所谓(你记得就行), 进入 文件夹 右击点击 选择 Git Bash Here
      • 输入 npm install hexo-cli -g –就是Hexo官网首页的哪一行命令行
      • 等待hexo 安装 (此步骤可能会花点时间,服务器在国外,解决方案就是挂代理)
      • 以上所有安装完,就可开始搭建博客

  • 第四步 :

    • 紧跟上一步 : 在Git Bash Here 中输入

      • hexo init 加上你的仓库名
        举例: 如果我的仓库名叫Gsiiiii 那么写法就是 Gsiiiii.github.io

    • 等待创建(hexo 会从云端下载)

      • 等待hexo 安装 (此步骤可能也会花点时间,服务器在国外,解决方案就是挂代理)

    • 创建完成,去Hexo官网选择主题,点击主题,进入该主题的 github 页面,

      • 网页首页会有一行 代码

      • 复制此代码 在 {git} 输入 hexo init + (复制的代码)

        • 等待下载

      • 下载完成去配置主题信息

        • 打开文件夹中的_config.yml 配置文件 (可以使用记事本进行修改或者其他工具)

        • 最重要的是修改使用的主题:

          • ——-> theme: diaspora(你使用的主题文件名)

        • 其他属性的修改

          • title: GuSu-Blog ——->(浏览器显示的标题)
          • author: Gs ——–(作者-写你自己的名字/昵称)
          • language: en ——->(语言)
          • timezone: Asia/Shanghai——>(时间/这里我设置的是 亚洲/上海)
          • url: gusu.xyz———>(这里修改为返回主页面的连接/自己的域名或者github 分配的子域名)

        • 进入 主题&图片 进行修改/替换 (默认情况下,主题内容里的Github 的地址和一些其他链接填写的是主题制作人本身的,所以需要 你自己进行修改)

          • themes 文件夹是你主题的文件夹
            • 进入文件夹,打开里面的_config.yml 文本文件
              • 你可以在这个文本中修改,Mp3 ,封面图
